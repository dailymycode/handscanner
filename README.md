

Installation
pip install -r requirements.txt

Execution
python main.py

Usage

Start the application

Place your left hand over the template

Hold your position for 2 seconds

The video will play automatically once verification is complete

Press ESC to exit

File Structure
main.py: Main program
hand_template.png: Hand template image
hand_landmarker.task: MediaPipe model
requirements.txt: Python dependencies
video/login.mp4: Video to play
